1 #breakpoints Breakpoints

Breakpoints for Towel.

Example

The following breakpoints are used by this theme and are automatically generated from toweltheme.breakpoints.yml:

  • Breakpoint Mobile

    • *The mobile breakpoint has no media query

  • Breakpoint Narrow ($breakpoint-narrow)

    • all and (min-width: 480px)

  • Breakpoint Normal ($breakpoint-normal)

    • all and (min-width: 768px)

  • Breakpoint Wide ($breakpoint-wide)

    • all and (min-width: 1280)

They can be used in scss like so:

.example-class {
  @include susy-media($breakpoint-narrow) {
    background-color: #ff5400;
  }
}

Other breakpoints may eventually be defined directly in scss/components/_breakpoints.scss if they do not need to be accessed from PHP or JS.
